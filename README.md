간단한 IOCP 게임서버 프레임 워크


## 브랜치

- master : 새 버젼 작성 중입니다.
- legacy : 기존에 작성했던 버젼입니다. 일단 코드 자체는 안정적으로 사용할 수 있는 형태의 프레임워크입니다.

## 빌드 요구사항
 - Windows 7+ & Visual Studio 2013+ (VS 2015 권장)
